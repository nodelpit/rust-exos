# Rust Exos

This repository contains exercises completed during my learning journey of Rust fundamentals, alongside reading *[The Rust Book](https://jimskapt.github.io/rust-book-fr/title-page.html)*.
## Objective

The goal is to strengthen understanding of core Rust concepts through hands-on practice, including:

- ownership and borrowing
- lifetimes
- slices and strings
- structs, enums, and impl blocks
- collections (Vec, HashMap)
- generics and traits
- error handling (Result, Option, panic)
- smart pointers (Box, Rc, RefCell, Arc, Mutex, RwLock)
- iterators and associated methods

## Exercise Organization

The exercises are organized by topic.

Each "Quest" series progressively introduces:
- increasing syntactic complexity
- more advanced concepts
- combinations of ideas from previous exercises

This approach helped reinforce my understanding of Rust by applying concepts in more realistic and interconnected scenarios.

## Further Practice

Throughout this learning process, quiz sessions were completed to reinforce and validate understanding of the main concepts covered.

## Outcome

These exercises helped me build a solid understanding of Rust fundamentals and its core principles.

## Methodology

The exercises were generated using a strict AI prompt designed as a learning and mentoring support tool.  
All code implementation was performed independently.
